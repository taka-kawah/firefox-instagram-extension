# AMO（addons.mozilla.org）申請用メモ

AMO に提出する際の掲載文・チェックリストの下書き。実際の提出は

## 名称
Instagram フォーカスフィルター / Instagram Focus Filter

## 概要（短い説明）
Instagram のホームから「おすすめ投稿」を消し、リール動画への導線を隠し、発見ページの
おすすめを非表示にして、フォロー中の投稿に集中できるようにします。

## 詳細説明（掲載用）
Instagram をブラウザで開くと、フォローしていない人の「おすすめ投稿」や、つい時間を溶かして
しまうリール動画への導線が目に入ります。本拡張機能は、それらを画面から取り除き、
**自分でフォローした相手の投稿に集中**できる状態にします。

主な機能:
- ホームのタイムラインから「おすすめ投稿」を非表示（多言語ラベル対応）
- 左メニューの「リール」への導線を非表示
- 発見（検索）ページで、検索バーは残しつつ、おすすめのサムネイル一覧を非表示
- 広告（Sponsored）を非表示にする任意オプション
- ツールバーのアイコンから、機能ごとにオン/オフ切り替え（設定は端末間で同期）

プライバシー: 個人情報・閲覧履歴は一切収集しません。保存するのはオン/オフ設定だけです。
（プライバシーポリシー: リポジトリ内 `PRIVACY.md`）

## カテゴリ
- 主: プライバシー & セキュリティ / または「ソーシャル & コミュニケーション」

## 権限の説明（審査時に問われた場合の回答）
- `storage`: トグル設定（どの機能をON/OFFにしたか）を保存・同期するため。
- `www.instagram.com` への content script: ページ上の表示を加工（おすすめ等を非表示に）するため。
  取得したページ内容を外部へ送信することはありません。

## データ収集の申告
- 収集データ: なし（manifest の `data_collection_permissions.required = ["none"]` と一致）。

## スクリーンショット（用意するもの）
> 実機（ログイン済み Instagram）での取得を推奨。`verification/screenshots/` の before/after も流用可。
1. ホーム: 適用前 → 適用後（おすすめが消え、フォロー投稿だけになる）
2. 左メニュー: 「リール」項目が消えている様子
3. 発見ページ: 検索バーは残り、おすすめグリッドが消えている様子
4. ポップアップ: トグルUI
